# PandoraTrader
A Trade Platform

这是一个国内期货交易平台，采用C++对CTP进行一层封装，提供友好的接口给策略交易模型。只需要利用策略模型进行策略开发即可，不用关心底层如何利用CTP进行下单撤单等控制。
